# Netflix Movies and TV Shows Data Analysis

## Dataset
Netflix Movies and TV Shows Dataset from Kaggle.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Steps
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Visualization
5. Insights Report

## Key Findings
- Movies are the largest category.
- Netflix content increased after 2015.
- USA produces a large amount of content.
